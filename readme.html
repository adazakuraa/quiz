このゲームの遊び方

＜手順１＞まず初めに、任意のcsvファイル（詳しくは下記の注意事項参照）を読み込ませて、そのファイル名が横に正しく表示されることを確認してください。重いファイルだと少々時間がかかることがあります。

＜手順２＞ルールを設定したければ「ルール設定」ボタンから、何○何×でクイズを行うかを設定してください。

＜手順３＞EnterキーかShiftキーを押すと、ファイル中の問題がランダムに出題されます。

＜手順４＞任意のキーを押すことで問題文の読み上げを止めることができます。

＜手順５＞問題文の読み上げが停止すると、プロンプトが表示されます。このプロンプトには押されたキーの名前が表示されるようになっています。

＜手順６＞答えを打ち込んだ上で「OK」を押すと正誤判定がなされます。答えを打ち込まなかった場合は自動的に誤答扱いになるので注意してください。

＜手順７＞プロンプトに対して「OK」を押すか、問題文が読み切られてから一定秒経つかすると、答えとこれまでの正解数・誤答数が表示されます。

＜手順８＞次の問題に移りたい場合は、EnterキーかShiftキーを押して画面をリセットした後、手順２に戻ってください。

＜その他１＞「速度設定」ボタンを押すと、問題文の表示されるスピードや、答えが表示されるまでの時間を変更することができます。

＜その他２＞「リセット」ボタンを押すと、それまでの各プレイヤーの正解数と誤答数がリセットされます。ただし、ルールはリセットされません。

＜その他３＞「名前登録」ボタンを押すと、それぞれのキーについて、使用する人の名前を登録することができます。

＜その他４＞Backspaceキーを押すと、得点が表示されなくなります。一人で勉強するときなどに使用してください。

    注意事項

  • 読み込むcsvファイルには、エクセルのA列に問題文、B列に答えが格納されているものを利用してください。この他のセルに値が入っていたり、問題文と答えが逆に格納されていたりすると正しくゲームを遊ぶことができない可能性が高くなります。

  • 回答の際に表示されるプロンプトにおいて、「キャンセル」を押すとその1秒後に問題文の続きが流れます。

  • 名前登録をしていない場合、正解数などは表示されません

  • 正誤判定について、回答として入力した文字列が答えの中に含まれていれば正解となります。そのため、諺が答えになる問題などではその最小単位を入力することを推奨します。ただし、表記揺れは未対応です。

  • 表記揺れなどのせいで誤答扱いになってしまった場合などは、正誤を表示するプロンプトに「1」を入力してください。その問題での誤答を正解として（あるいは正解を誤答として）記録し直します。また、このプロンプトに「2」を押した場合は、この問題を無かったこと（スルー）として扱います。

  • その他のバグや設定等を変更したければ「quiz.js」のソースコードを修正してください。
